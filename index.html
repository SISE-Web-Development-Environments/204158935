    • About Me 


    
        קוראים לי אשר ואני בן 76. ב1949 עליתי עם הוריי ושלושת אחיי הקטנים לארץ מאיזמיר, טורקיה.
        בארץ הכרתי את אישתי אינס, נולדו לנו ארבע בנות בשמות: רונית, ליאורה, אילנית ושושנה.
        השתתפתי במלחמת ששת הימים ובמלחמת יום הכיפורים. כאשר אילנית נולדה במהלך מלחמת יום הכיפורים, לא ידעתי על כך במשך שלושה שבועות!
        אני דובר עברית וספליונית.
        במשך חיי עבדתי בהרבה עבודות כאשר העיקרית שבהן היא כטבח לסוהרי שירות בתי הסוהר. התחביבים העיקריים שלי הם לבשל ולדוג.
        כיום, יש לי 14 נכדים ושני נינים. את רוב זמני היום אני מעביר בסידורי הבית הסתובבות בשווקים וישיבה עם חברים בשכונה.
        אני משקיע המון זמן בבילוי עם נכדיי האהובים ולכן קשה לי מאוד בבידוד שהוטל עלינו בימים אלו!

    Q&A

    1. 3 main diffrences between web 1.0, web 2.0 and web 3.0:

    while web 1.0 purpose is for read only, web 2.0 is read and write web and web 3.0 is read, write and exectute web.
    web 1.0 focus was for companies, web 2.0 focus was the general community and web 3.0 focus on the indevidual.
    web 1.0 Content is served from the server’s file-system, web 2.0 Information flows between site owner and site users by means of evaluation & online commenting, while web 3.0 computers can distinguish information like humans in order to provide faster and more relevant results.

    2. diffrence explenation by use case example:"find side effects of drugs"

    web 1.0- The user can only view drug companies websites, scroll and search about the drug and it's side effects. He will need to know the drug company URL.
    web 2.0- The user can type the drug company name and enter it's website. here he can search for side effects of the specific drug.
    web 3.0- The user can type keywords like the drug name and side effect, and the web(search engine) will present relevant pages.
