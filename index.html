    קוראים לי אשר ואני בן 76. ב1949 עליתי עם הוריי ושלושת אחיי הקטנים לארץ מאיזמיר, טורקיה. בארץ הכרתי את אישתי אינס, נולדו לנו ארבע בנות בשמות: רונית, ליאורה, אילנית ושושנה. היום אני סבא ל14 נכדים ושני ושני נינים. אני דובר עברית וספליונית. השתתפתי במלחמת ששת הימים ובמלחמת יום הכיפורים. במלחמת יום הכיפורים ביתי אילנית נולדה ולא ידעתי על כך שלושה שבועות. במשך חיי עבדתי בהרבה עבודות כאשר העיקרית שבהן היא כטבח לסוהרי שירות בתי הסוהר. התחביבים העיקריים שלי הם לבשל ולדוג. כיום, אני בפנסיה ולכן אני מעביר את רוב זמני בסידורי הבית, עם נכדיי וגם בצפייה בסרטים בטלוויזיה.

    Q&A

    1. 3 main diffrences between web 1.0, web 2.0 and web 3.0:

    while web 1.0 purpose is for read only, web 2.0 is "writable" and web 3.0 is portable and personal. web 1.0 focus was for companies, web 2.0 focus was the general community and web 3.0 focus on the indevidual. web 1.0 Content is served from the server’s file-system, web 2.0 Information flows between site owner and site users by means of evaluation & online commenting, while web 3.0 computers can distinguish information like humans in order to provide faster and more relevant results.

    2. diffrence explenation by use case example:"find side effects of drugs"

    web 1.0- The user can only view drug companies websites and search about side effects. He will need to know the drug company URL.
    web 2.0- The user can enter the drug company website, and search for side effects of the specific drug.
    web 3.0- The user can type keywords like the drug name and side effect, and the web will present relevant pages.
